相对于 [VUE CLI](https://cli.vuejs.org/zh/)，YMA 的优点

-   开箱即用，一站式解决工程化所面临的问题
-   支持自定义命令扩展
-   支持 Webpack 生态周边扩展
-   [presets & plugins] 简单完成已有配置的应用

## 使用

```sh
npm install yma-cli

yma <cmd> [options]
```

## 发包

```sh
# 升级版本
npm run version

# 发布
npm run publish
```
